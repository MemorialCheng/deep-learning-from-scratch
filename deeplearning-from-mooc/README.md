# 深度学习实战

本课程代码分为两个版本，分别针对python2.7和python3.6。

目前已有的代码结构如下：


- py2
	- 02-nn-basic: 第二章 神经网络入门
	- 03-conv-basic: 第三章 卷积神经网络
	- 04-advanced-conv: 第四章 卷积神经网络进阶
	- 06-style-transfer: 第六章 图像风格转换
	- 07-rnn-text-classification: 第七章 循环神经网络与文本分类
- py3
	- 02-nn-basic: 第二章 神经网络入门
	- 03-conv-basic: 第三章 卷积神经网络
	- 04-advanced-conv: 第四章 卷积神经网络进阶
